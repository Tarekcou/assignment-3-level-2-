# PostgreSQL Assignment - B7A3

## 📌 Assignment Overview

This repository contains the solution for the **B7A3 PostgreSQL Assignment**. The assignment demonstrates fundamental and intermediate SQL concepts using PostgreSQL, including database creation, data manipulation, filtering, joins, grouping, subqueries, constraints, and aggregate functions.

## 📂 Project Structure

```
.
├── schema.sql          # Database schema (CREATE TABLE statements)
├── insert_data.sql     # Sample data insertion queries
├── queries.sql         # Solutions to all SQL queries
├── answers.md          # Answers to theoretical questions
└── README.md
```

## ▶️ How to Run

1. Create a PostgreSQL database.
2. Execute the SQL files in the following order:

```sql
-- 1. Create tables
schema.sql

-- 2. Insert sample data
insert_data.sql

-- 3. Run assignment queries
queries.sql
```

3. Read the theoretical answers in `answers.md`.

## 📁 Files Description

| File              | Description                                   |
| ----------------- | --------------------------------------------- |
| `schema.sql`      | Creates all required database tables          |
| `insert_data.sql` | Inserts the provided dataset                  |
| `queries.sql`     | Contains solutions for all assignment queries |
